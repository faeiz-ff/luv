
# luv Programming Language

`luv`, a simple programming language with soft looks and strong types. The syntax is inspired by [Go](go.dev), [Rust](rust-lang.org), [Zig](ziglang.org) and [Python](python.org).

`luv` is an interpreted language that combines a Object-Prototype runtime model like [JavaScript](developer.mozilla.org/en-US/docs/Web/JavaScript) and Static Type System that is inspired by [TypeScript](typescriptlang.org) and Rust.

This is only a hobby project of mine to learn low level stuff.
