
# luv Programming Language

`luv`, a simple programming language with soft looks and strong types. The syntax is inspired by [Go](https://go.dev), [Rust](https://rust-lang.org), [Zig](https://ziglang.org) and [Python](https://python.org).

`luv` is an interpreted language that combines a Object-Prototype runtime model like [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript) and Static Type System that is inspired by [TypeScript](https://typescriptlang.org) and Rust.

This is only a hobby project of mine to learn low level stuff.
100% human written. I take great pride in my bugs and segfaults.

Things that I want `luv` to achieve:

- a [Raylib](https://raylib.com) FFI
